## 🌌 Overview

This project explores a simple but striking idea:

> A single geometric entropy law, parameterized by an effective dimension `d_eff`, can reproduce:
- Particle mass spectrum (leptons, quarks, hadrons, bosons)
- Cosmological energy scales (Planck, GUT, inflation, etc.)

The entire structure is generated from:

S(d) → ΔV → I_eff² → L* → m

## 🧠 Core Idea

Entropy is defined geometrically as:

S(d) = α_geom · A_d(R) / l_Pl^(d−1)

Where:
- `d_eff` = effective geometric dimension
- `α_geom` = universal normalization constant
- `A_d(R)` = hyperspherical area in dimension d
- `l_Pl` = Planck length

Mass emerges from an information cascade:

Entropy → Vacuum energy → Information scale → Length scale → Mass

## 🔬 What the script does

The script:

1. Defines a geometric entropy model
2. Converts entropy into energy/mass
3. Scans `d_eff` over a continuous range
4. Matches:
   - Particle masses (PDG values)
   - Cosmological energy scales
5. Outputs best-fit `d_eff` for each physical system
